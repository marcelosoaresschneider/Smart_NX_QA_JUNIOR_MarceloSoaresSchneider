# Smart NX - QA Jr - Robot Framework + SeleniumLibrary (OrangeHRM)

Projeto de automação de testes Web simples com Robot Framework e SeleniumLibrary. Aplicando cenários em escritos em Gherkin (Dado/Quando/Então) no site OrangeHRM. Inclui testes de Login, do módulo PIM, Cadastro de “Employee”, Filtros de “Employee” e Exclusão de “Employee”.


## Estrutura do repositório do projeto

- `tests/`  
  Contém os arquivos `.robot` (casos de teste).
- `resources/`  
  Contém os arquivos `.resource` (keywords, variáveis e locators).
- `results/`  
  Pasta gerada automaticamente ao executar os testes (logs e relatórios).
